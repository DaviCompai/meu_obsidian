# Onde programar?

### No computador
O Visual Studio 2022 pode ser pesado de mais para alguns dispositivos mais antigos. Caso este seja seu caso, eu recomendo o uso do visual studio code.
Tutorial de instalação e configuração do c#: https://www.youtube.com/watch?v=S5HawDwjuUY
### No celular
para android: [# C# Shell .NET IDE](https://play.google.com/store/apps/details?id=com.radinc.csharpshell) 

para iphone: [codesnack ide](https://apps.apple.com/br/app/codesnack-ide/id1397424959) na opção de C# .
Para isso, clique em "new project" --> "from template", escreva o nome do projeto, vá na opção de c# e escolha "c#(Basic(.NET))" --> "create" e espere por alguns segundos. Após isso, você poderá acessar seu código em Program.cs, pelo botão de pasta, no canto superior direito.
